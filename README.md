**Live Demo:** [ใส่ลิงก์ GitHub Pages ที่ได้มาตรงนี้]

## 📌 About The Project
CyberSentinel is a modern, real-time web application designed to enhance cybersecurity awareness. It allows users to scan their passwords against global data breach databases (via HaveIBeenPwned API) and generate mathematically secure passwords on the fly.

## ✨ Key Features
- **Real-time Breach Scanning:** Uses SHA-1 hashing and RESTful API to securely check passwords without exposing plain text.
- **Password Strength Meter:** Evaluates password complexity in real-time.
- **Secure Password Generator:** Custom algorithm to generate 8-32 character passwords with custom symbols, numbers, and cases.
- **Modern UI/UX:** Built with Tailwind CSS, featuring a Cyberpunk/Hacker theme with glassmorphism effects.

## 🛠️ Built With
- HTML5 / CSS3
- Tailwind CSS
- Vanilla JavaScript
- HaveIBeenPwned API
